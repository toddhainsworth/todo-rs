# Install
```bash
$ git clone git@github.com:toddhainsworth/todo-rs.git ~/todo-rs
$ cd ~/todo-rs && cargo install
$ todo -h
```

Note; eventually i'll release standalone binaries for Linux and OSX (sorry Windows-folk)
